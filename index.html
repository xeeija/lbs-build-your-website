Website Building Basics

Every website uses the HTML (Hypertext Markup Language) and CSS (Cascading Style Sheet) for their structure and layout respectively.
Most of them also use JavaScript to implement functionaly into the page.
  • HTML
  • CSS
  • JS

  • HTML Elements

    Elements have a name and are enclosed in angle brackets. 

    <body>...</code>

  • Head

    The head element contains all relevent meta information as well as links to stylesheets.

  • Body

    The elements in the body define how the structure of the website. These are the visible elements.

  • HTML File

    Every HTML file starts with a tag, that indicates the format. 

    <!DOCTYPE html language="en">

  • Common Elements

    Most used elements are the container <div>, paragraph <p>, heading 1 to 6 <h1> and link <a>.

  • Class and ID

    Elements can have an id attribute, which must be unique, and a class attribute, that can be shared.

Cascading Style Sheets

  • Styling

    Styling options can be applied on a number of different properties of an HTML element, for example color, size and positioning.

  • Separation

    CSS separates layout from content and structure, so all pages easily have the same style.

  • Limits of CSS

    For large projects, tools like Sass or Less are used, to introduce missing features like nesting and variables.

  • Selector

    Styling options can be applied on a variety of properties. A CSS selector is used to specify what should be affected.

  • Selector Types

    Important selectors are the element type, the .class of the element and the unique #id.

  • Combining Selectors

    Multiple selectors can be grouped together with a comma. Different selector types can also be combined.

What is the future of Website-Building

There are a lot of opinions regarding the future of AI, but they basically agree on the following points:

  • Users will determine

    Users will visualize and describe the content to be displayed. Based on the technology of cookies, they already do this partially.

  • Digital Footprint

    AI will recognize who you are, what you like, who you are connected to and lots of other things based on your digital footprint.

  • Engaging is king

    Website will be deeply engaging, which means that the experience feels more like there is a human behind a screen.

  • Virtual Reality

    VR will also be an important technique in web design in terms of engaging with a user.

  • Transformation

    Web design will transform from being a brochure to being an interactive storytelling-machine.
